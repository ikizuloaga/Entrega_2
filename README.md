## Estructura del proyecto

En el directorio `src` se encuentran el archivo `.py` de la práctica.

En el directorio `notebooks` se encuentra el archivo `.ipynb` en el que se importan y se ejecutan las funciones.

## Requisitos

* Python 3.x
* Jupyter Notebook

##  Instalación de dependencias

```bash
pip install -r requirements.txt
```

##  Ejecución

1. Ubicarse en la carpeta raíz del proyecto

2. Ejecutar:

```bash
jupyter notebook
```

3. Abrir el archivo:

```
notebooks/ejercicio10.ipynb
```

4. Ejecutar todas las celdas
